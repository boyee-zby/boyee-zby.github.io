体成分测量对比及优缺点辨析——运动生理学作业

本文最后更新于：2020年10月10日 晚上

写在前面

该文章是我在10.9日运动生理学课堂展示的内容（作业），文章内容仅供参考；

体成分定义

人体成分(body composition）是指人体中水分、肌肉、脂肪、无机盐等各组分的含量及其在人体总质量中所占的百分比，通常可分为脂肪组织和非脂肪组织两部分，其质量和分布情况与各年龄阶段人群的健康状况密切相关。

随着人体成分测量和分析在临床上的应用，不仅可以了解人体基本健康状态和全身营养状况，对多种疾病的诊断、治疗、监测及预后进行评估，将疾病风险分级，还可以为相关疾病的预测提供信息，减少相关疾病的发生，促进健康，提高生活质量。
此外，测量体成分还可以估算健康体重，为训练计划提供指导，监控儿童青少年的成长发育，追踪老龄化和疾病的相关变化。
在临床上，人体成分测量的研究涵盖了物理学、生物学、医学和公共卫生学等，一般分为局部测量法与全身测量法，根据其技术差异，各检测方法的优缺点、适用情况也各有不同。

脂肪分类

  • 褐色脂肪（精华）
    • 褐色脂肪是为日常活动所供能的，在肌肉、神经细胞、骨髓、肠道、心脏等处分布、男性有3%-5%，女性有8%-15%（在胸部和子宫上）。
  • 白色脂肪（非精华）
    • 白色脂肪是为了储存能力，大多为动物脂肪，有着缓冲、隔离保暖的作用、内脏脂肪（分布在器官周围）、皮下脂肪（在皮肤下面）

脂肪重量

脂肪重量大约占体重的15%，去脂肪后体重大约占总体重的85%；「水=73.8%、蛋白质=19.4%、矿物质=6.8%」

按测量区域选定适合的测量方法

人体成分全身测量法

人体成分测量的全身测量法包括生物电阻抗法、双能X线吸收法、水下称重法、空气置换法、同位素稀释法、钾同位素测量法、近红外线吸收法、三维人体扫描法等。根据其方法的技术适用情况、临床应用情况，以及实际需求等情况不同，适用的人体测量方法也不尽相同，但综合分析更有关注价值的是生物电阻抗法。

研究表明 | 生物电阻抗法作为人体成分测量方法更为可靠，其灵敏度也更适用于儿童超重和肥胖的筛查。与皮褶厚度法比较，生物电阻抗法更加精确；与水下称重法比较，生物电阻抗成本和技术难度更低，安全无创伤，更适合于大规模群体研究和流行病学调研，对诊断肥胖或营养不良、检测慢性病患者营养状况等更具有重要价值。

生物电阻抗法

原理：细胞的液体重与电阻抗值成密切正相关，可根据电阻抗值直接计算出细胞内外水重。此时，超过200kHz频率的电力穿过细胞模式，即可测出人体总水分，推断出各项人体成分指标。
「通过低强度电流测量（脂肪导电性差，增加了电阻抗，身体含水量高。减少了电抗阻），通过身体的水含量估算去脂体重、可以通过这种方法间接估算去脂体重和身体含水量。」

生物电阻抗法可测得细胞外液、细胞内液、体内总水分、体脂肪、体蛋白、肌肉、瘦体重、矿物质8种人体成分，可推算出11项指标，有脂肪百分比、肥胖程度、BMI、基础代谢率、标准肌肉、标准体重、体重控制、脂肪控制、肌肉控制、目标体重和水肿系数。


人体成分分析仪

人体成分分析报告

  • 优点
    结果精确、操作简便、安全无创、多用途、成本低等
  • 缺点
    易受体液变化，如饮水、膳食、腹泻及运动等影响；
  • 误差来源
  1. 设备（不同型号设备间的差异）
  2. 受试者（水合状态）
  3. 测试者技术（受试者身体位置或电极摆放不正确）
  4. 环境（电阻抗与皮肤温度呈负相关）
  • 适用情况
  1. 儿童超重与肥胖筛查；
  2. 大规模群体研究及流行病学调查；
  3. 对肥胖、营养不良、慢病患者营养监测有重要价值。
  • 注意事项
    • 在测试前要进行水合状态控制；
  1. 测试前4h不进食、测前12h内无运动、测前48h避免酒精摄入、不得服用利尿剂；
  2. 遵循测试前指南、采用标准步骤（被测者身体位置、手臂和双腿伸直、电极位置摆放正确），选用正确的生物电阻抗公式。
水下称重法

原理：该方法是建立在脂肪组织与非脂肪组织的密度测量基础上，用人体在水下减轻体重计算身体密度，间接测量体脂率和去脂体重。(身体质量-体重秤；身体体积-水下称重；密度=身体质量/身体体积）

操作流程及注意事项：

  1. 遵循测试前的指南；
  2. 校准设备并使用标准测试步骤；
  3. 受试者必须完全沉入水中并尽力呼气；
  4. 三次误差不超过100g。

  • 优点
    比较和评定其它方法的“金标准”
  • 缺点
    设备复杂、技术难度较大、耗时较长，应用具有局限性；
  • 适用情况
  1. 适用于比较、评定其他方法；
  2. 大规模群体测定不适用；
  3. 体质较弱及患有呼吸系统疾病的人不适用。
双能X线吸收法
  • 原理：两种不同能量的X线照射的生物体上的，由于X线透过不同组织的衰减程度不同（衰减比例取决于组织的厚度、密度及构成），可根据不同衰减率得到全身骨密度、脂肪、瘦体重、内脏脂肪等。

  • 优点
    准确性高于水下称重法和钾同位素测量法，操作简便、结果易分析、测量者未感不适

  • 缺点
    设备昂贵、工作人员技术要求较高、临床使用受限

  • 适用情况

  1. 对身体成分与疾病关系研究具有实用价值；
  2. 临床适用于各年龄人群，但受年龄、性别、体型、肥胖程度及健康状况等影响；
  3. 可作为标准方法，验证其他体成分测量方法的准确性。
  • 注意事项
    需要可操作X射线技术人员进行操作，且受试者在测试过程中保持仰卧位静止
近红外线吸收法
  • 原理：由于脂肪和水分对红外线的吸收带不同，用受光传感器捕捉从测定部位反射出来的散乱吸收光，计算该部位的脂肪厚度，再加上身高、体重参数，可利用公式推测身体脂肪率。
  • 优点
    简单、迅速、无创
  • 缺点
    可靠性尚需有效证实，对重度肥胖者容易评价过低，不能充分反映内脏脂肪问题。
  • 适用情况
    人体成分进行粗略分析的场合
三维人体扫描法
  • 原理：扫描仪捕获身体表面形态，应用数字技术描绘出表面集合图形，通过计算机计算出3D身体图像，包括整体和局部的影像，腰围、臀围，身体各段体积和体表面积。

  • 优点
    安全、方便、易于操作、不接触受试者身体
  • 缺点
    精准性尚未得到充分验证，不能区分内脏脂肪和皮下脂肪。
  • 适用情况
  1. 大规模流行病学调查研究；
  2. 患者治疗效果的评估与检测。
人体测量法
  • 原理：通过测量人体的身高、体重、腰围、臀围、腰臀比等来确定身体质量、大小、形态、肥胖程度等。

  • 优点
    简单、安全、廉价
  • 缺点
    不能真实反映体内脂肪堆积和分布情况、不能区别腹部皮下脂肪和内脏脂肪，与人群的性别、年龄、种族、地域、肥胖程度等因素相关。
  • 适用情况
    患者恢复情况评估和临床参考指标
人体测量学与疾病风险分级的关系：
  • BMI能够预测出二型糖尿病等疾病；
  • 腰围也能够预测出肥胖等健康风险：男性>102cm,女性>88cm则说明有风险；
  • 腰臀比能够反映出肥胖症、身高和腰围比可以预测出肥胖与相关病症的关系。

适用于科研的方法

空气置换法

  • 原理
    原理与水下称重法相同，测量者进入容器，感受器计算压力测出人体排出的空气里，从而推算出脂肪组织与非脂肪组织的基准值。「理想气体的体积与气体的压强成反比」
    计算公式：p1/p2=v2/v1；（适用双仓系统和博伊尔定律）
    「身体体积经过肺内气体体积校正、通过身高和体重估算体表面积」

  • 优点
    可精准分析人体成分，包括密度、体质含量、体脂率等（比水下称重更快操作更简单）

  • 缺点
    精准性需要待验证，设备复杂、不适用于大样本研究，目前应用较少

  • 注意事项
    为保证准确性需穿戴紧身衣和泳帽、在测试过程中严格遵循校准步骤

同位素稀释法

  • 原理
    脂肪中不含水，非脂肪中水含量较为恒定，一般通过标记法测定身体中总水量，即可估计体脂量和非脂肪组织量。
  • 优点
    准确、快速、无创人体脂肪含量
  • 缺点
    技术难度高、价格昂贵

    钾同位素测定法

  • 原理
    体内钾主要分布在非脂肪组织中，且浓度恒定不变，因此可通过测定身体中钾的总含量，推算非脂肪组织和脂肪组织的量。
  • 优点
    人体成分较为有效的测定方法
  • 缺点
    国内尚未报道，国外只用于试验

人体成分局部测量法

局部测量法包括皮褶厚度法、超声检测法、计算机断层扫描法以及核磁共振法。通过对身体局部检测人体成分来推断身体健康情况。

皮褶厚度法
  • 原理：人体脂肪主要分布在皮下、大网膜及肠系膜等处，其中约40%-60%贮存在皮下组织，在身体许多部位，皮肤及皮下脂肪疏松地附着于其下的组织上，因此可以通过皮脂厚度仪测量出皮肤褶皱的厚度，从而用公式推算全身脂肪含量。

  • 优点
    无损伤、操作简单、以其轻便易于携带
  • 缺点
    由于仪器、设备、个体差异等不同，应用中存在测量精度和稳定性难控制的现象
  • 适用情况
    大样本群体测量
计算机断层扫描法
  • 原理：X线穿过人体不同组织的衰减程度不同，据此可识别骨骼、肌肉、脂肪等不同机体组织。计算机断层扫描法能对各部分组织直观成像，可准确测量脂肪、肌肉、内脏等组织的面积及含量。

  • 优点
    能够帮助研究者更加深入的了解人体成分与健康的复杂关系，并对脂肪肝水平评估具有特异性。
  • 缺点
    价格昂贵、辐射量大，难以广泛临床使用
  • 适用情况
    评估腹部脂肪分布和预测腹型肥胖较准确
核磁共振法
  • 原理：人体中的H质子在强磁场中受到射频脉冲的继发，产生核磁共振现象，经过空间编码技术，通过计算机形成图像，根据扫描层面脂肪组织的面积、体积推算总体脂肪和局部脂肪的含量。

  • 优点
    无放射性
  • 缺点
    价格昂贵、耗时长、受试者易产生幽闭恐惧症
  • 适用情况
  1. 人体成分测量的“金标准”；
  2. 主要用于医学研究。
超声检测法
  • 原理：将人体各部位各层组织回声通过探头转化为声像图，间接反映人体各部位组织结构。
  • 优点
    无创、价廉、简便、精准、可靠
  • 缺点
    产后女性、青少年、腹水患者的准确度低，且与肠腔是否充盈与充盈程度有关。
  • 适用情况
    ①减肥治疗效果评价；
    ②对糖尿病、高脂血症、脂肪肝、冠心病等疾病危险因素评估与监测。

写在最后

人体成分测量有助于了解人体营养水平及健康情况，且对多种疾病的争端、治疗、检测和预后评估有重要价值。局部检测法通过局部检测推断人体健康水平，但据其操作性、稳定性等在临床营养上并不广泛，而全身测量法营养部分方法应用相对较多，根据临床和各方文献综述评定，生物电阻抗法更具有长远的应用价值和临床意义。

目前，临床营养上，采用生物电阻抗技术的人体成分分析仪已应用于孕期营养管理、中小学智慧营养系统建设、慢性疾病营养管理、企业团队健康管理、老年康复营养门诊建设、肿瘤营养管理等多个领域。在科研、医疗临床上，也为各类人群的营养管理提供了科学依据。
